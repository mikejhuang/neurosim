MasterConfBuilder
Use the NEURON GUI to choose your synapse locations and convert these locations to a masterconf file. 

Instructions:
1. Open neuron model in NEURON GUI
2. Go to Tools > Point Processes > Managers > Point Group
3. Remove the Iclamp present
4. Go to New > Alpha Synapse
5. Press Copy to create as many synapses as needed
6. Select each synapse and click the position on the neuron to place them
7. Go to File > save session
8. Run MasterConfBuilder in the terminal by typing sh MasterConfBuilder.sh
9. Follow instructions on the screen
